🎓 Student Record System (C++)

Console-based application for managing student records using a Binary Search Tree (BST).

---

📌 Description

This project is a simple student management system written in C++.
It allows users to store, search, and manage student records efficiently using a Binary Search Tree data structure.

---

🚀 Features

- Add a new student
- Find student by matriculation number
- Search student by name
- Count total enrolled students
- Display all student records
- Remove a student
- Save data to file
- Load data from file

---

🛠️ Technologies Used

- C++
- Binary Search Tree (BST)
- File Handling

---

▶️ How to Run

1. Compile the project:

g++ student_record_system.cpp main.cpp -o student_app

2. Run the program:

.\student_app.exe
